# Wikipedia Content Analyzer

A Python tool utilizing Wikipedia's API to retrieve and analyze article content.

## Features

- Data retrieval: Utilizes Wikipedia's API to fetch content for a specified article.
- Data cleaning: Extracts meaningful content, removes irrelevant data, and formats information for analysis.
- Text processing: Conducts tokenization and lowercasing of words for effective data processing.
- Term frequency model: Creates a model to analyze word occurrences, laying the foundation for search engine functionality.
- Data visualization: Uses matplotlib to present the 20 most frequent words in the article.
- Stop words removal: Implements a function to filter out insignificant words, enhancing content relevance.
- Improved classification: Visualizes the 25 most frequent words after stop words removal to demonstrate enhanced content classification.
